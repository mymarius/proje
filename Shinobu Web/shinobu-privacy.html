1. Legal Framework and Scope

1.1. Legal Framework

This Privacy Policy and Data Protection Guidelines have been prepared in accordance with:

  • California Consumer Privacy Act (CCPA)
  • California Privacy Rights Act (CPRA)
  • General Data Protection Regulation (GDPR)
  • Children's Online Privacy Protection Act (COPPA)
  • Federal Trade Commission Act
  • State Privacy Laws including Virginia CDPA, Colorado CPA
  • Discord Platform Terms of Service and Policies
  • Applicable Federal and State Data Protection Laws

1.2. Scope and Purpose

This policy establishes the procedures and principles governing data processing activities performed by Shinobu Bot ("Service Provider") through the Discord platform.

2. Definitions and Abbreviations

2.1. Key Definitions

  • Bot: Automated software application operating on the Discord platform
  • Personal Information: Information that identifies, relates to, or could reasonably be linked to a particular individual
  • Data Controller: Legal representative of Shinobu Bot
  • Server Owner: User with legal administrative authority over a Discord server
  • API: Application Programming Interface
  • VPS: Virtual Private Server

2.2. Technical Definitions

  • Prefix: Special character sequence used to trigger bot commands
  • Server ID: Unique identifier assigned by Discord platform
  • Token: Bot authentication key
  • Endpoint: API access point
